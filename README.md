# Netflix Content Analysis

This project explores a Netflix titles dataset using Python, pandas, and Plotly. It includes data cleaning, exploratory analysis, and interactive visualizations for content type, genre, country, and rating distribution.

## Files
- `netflix_analysis.ipynb` — main notebook with the analysis and plots
- `netflix_titles_nov_2019.csv` — dataset used for the analysis
- `plot_content_type_pie.png` — content type split pie chart
- `plot_content_type_bar.png` — content type trend bar chart
- `plot_genres.png` — top genres bar chart
- `plot_countries.png` — top countries bar chart
- `plot_geo_countries.png` — geographic distribution choropleth
- `plot_ratings.png` — rating distribution bar chart

## How to Run
1. Install the required Python packages:
   - pandas
   - numpy
   - matplotlib
   - seaborn
   - plotly
   - kaleido (for saving Plotly charts as images)
2. Open and run the notebook in Jupyter.

## Key Insights
- Movies make up the majority of the catalog.
- Drama is one of the dominant genres.
- The United States is the leading content-producing country.
- Mature-rated content is strongly represented.
